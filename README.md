### Clone the project

```bash
$ cd ~
$ git clone git@github.com:maximepersonnic/dotfiles.git
$ cd dotfiles
$ find ~/dotfiles -name '*.sh' | xargs -I file chmod u+x file
```

### Brew

```bash
$ brew/brew.sh
```

### Git

```bash
$ git/git.sh
```

### SublimeText

```bash
$ SublimeText/sublime.sh
```

### ZSH

```bash
$ zsh/zsh.sh
```

### SSH

```bash
$ cp ssh/config ~/.ssh/config
```

### VS Code

```bash
$ vscode/vscode.sh
```
