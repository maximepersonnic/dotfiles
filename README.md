### Clone the project

```bash
$ cd ~
$ git clone git@github.com:maximepersonnic/dotfiles.git
$ cd dotfiles
$ find ~/dotfiles -name '*.sh' | xargs -I file chmod u+x file
```

### Brew

```bash
$ brew/brew.sh
```

```bash
$ git/git.sh
```
